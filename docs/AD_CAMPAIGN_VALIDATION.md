# 네이버 검색광고 검증 캠페인 (보류 중 — 재개 시 이 문서부터)

> 상태: **패키지 완성, 집행 대기** (2026-07-05 보류). 광고 계정 있음. 예산 승인됨(2주 ~14만원).
> 목적: 비치헤드 검증 — 훅 A/B(안전 vs 자격필터) + 가입 전환. 전화 0통 검증기계의 1부품.

## SERP 정찰 결과 (2026-07-05, insane-search로 실측)
- `투찰가 계산`: **파워링크 광고 0건.** 오가닉 1위 디마툴즈(자체 계산기 페이지). **bideasy.kr 오가닉 노출 중**(6/16 SEO 작동).
- `나라장터 입찰`: 파워링크 0건(이번 조회 기준). 공식·뉴스 위주.
- 함의: 도구 키워드 광고 선반 비어있음 → CPC 바닥값 예상, 하루 1만원=클릭 50~100 가능. 인포21C·케이비드·비드톡톡 검색광고 부재(전화영업 중심).
- 캐비엇: 광고 노출은 시간대·세션 변동 → 키워드 도구 '경쟁정도'로 최종 확인.

## 캠페인 패키지 (광고 시스템 입력용)
- 구조: 캠페인 `BidEasy-검증` → 광고그룹 1 → 키워드 ~10 → 소재 2(A/B 로테이션)
- 키워드: 투찰가 계산 / 투찰가 계산기 / 투찰가 산정 / 낙찰하한율 계산 / 낙찰하한가 계산 / 적격심사 계산 / 적격심사 계산기 / A값 계산 / 예정가격 계산 / 투찰율 계산
- 소재 A(안전 훅): 제목 `무효입찰 막는 계산기` / 설명 `투찰가·낙찰하한선을 1원 단위로. 무효·적자 없이 안전하게 투찰하세요.`
  URL `https://bideasy.kr/calculator?utm_source=naver&utm_medium=cpc&utm_campaign=validation&utm_content=safety`
- 소재 B(자격필터 훅): 제목 `입찰자격 자동 확인` / 설명 `면허·지역·실적으로 넣을 수 있는 공고만. 투찰가 계산까지 화면에서 바로.`
  URL 동일하되 `utm_content=filter`
- 설정: 수동입찰 150원(상한 300원) · 일예산 10,000원 · 전국 · PC+모바일 · 2주
- 측정: first-touch UTM(`bd_attr`)+GA4 자동. 소재별 가입전환 = `/admin/stats/attribution`의 utm_content.
- 판정: 소재 CTR 차 = pain 순위 / 가입 전환 = 훅 깊이 / 설문(미구현) = 비치헤드 적합도.

## 재개 시 남은 작업
1. 광고 시스템에 패키지 입력 + 키워드 도구로 경쟁정도·검색량 확인 (사용자)
2. 가입 직후 1문항 마이크로 설문 구현 ("주로 어떤 입찰? 월 몇 건?") (코드)
3. 커뮤니티 질문글 초안 (전기공사 카페/밴드용, 서면 인터뷰 대용)
