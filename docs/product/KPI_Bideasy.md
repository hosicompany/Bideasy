## 🎯 North Star Metric

## 📊 1. 제품/기술 KPI

| 지표 | MVP (Phase 1) | Phase 2 | Phase 3 |
|------|--------------|---------|---------|
| **예측 정확도 (MAE)** | ±3% 이내 | ±2% 이내 | ±1.5% 이내 |
| **예측 구간 적중률** | 70% | 80% | 85% |
| **API 응답 속도** | <2초 | <1초 | <0.5초 |
| **시스템 가동률** | 99% | 99.5% | 99.9% |

## 📅 5. Phase별 마일스톤

### Phase 1 (MVP) - 4주

| 주차 | 마일스톤 | 성공 기준 |
|------|----------|----------|
| 1주 | API 연동 + 데이터 수집 | 1만건+ 데이터 확보 |
| 2주 | 예측 모델 v1 개발 | 백테스트 MAE ±3% |
| 3주 | 백엔드/프론트 통합 | E2E 테스트 통과 |
| 4주 | 베타 출시 | 가입자 50명 |

### Phase 2 - +8주

| 마일스톤 | 성공 기준 |
|----------|----------|
| 온톨로지 라이트 적용 | 설명 가능한 예측 제공 |
| ML 모델 도입 | MAE ±2% 달성 |
| 유료 플랜 런칭 | 전환율 5% |

### Phase 3 - +12주

| 마일스톤 | 성공 기준 |
|----------|----------|
| 앙상블 모델 | MAE ±1.5% 달성 |
| 풀 온톨로지 | 다차원 분석 제공 |
| 월 매출 300만 | MRR 달성 |
