# BidEasy 운영 위임 런북 (Hermes 에이전트)

> 로컬 Hermes 에이전트에게 BidEasy "전체 운영"을 위임하기 위한 **권한 계약서**.
> 명세된 것만 위임된다. 등급은 대표 승인으로만 변경.
>
> **상태: 확정 (2026-06-20).** 다음 = Phase 1 "매일 운영 브리핑" 에이전트(🟢 읽기전용)부터.

## 권한 3등급
- 🟢 **AUTO (루틴)** — 에이전트가 직접 처리. 사후 감사 로그만. 되돌릴 수 있거나 읽기 전용.
- 🟡 **APPROVE (중요)** — 에이전트가 *초안/제안* 생성 → **대표 1클릭 승인** 후 실행. 외부로 나가거나 돈 안 닿는 변경.
- 🔴 **HUMAN (핵심·비가역)** — 사람이 직접. 에이전트는 *탐지·리마인드·초안*까지만, **실행 절대 금지**. (돈·시크릿·권한·삭제·공개발행)

> 🔴는 협상 불가. 라이브 결제 시스템에 자율 에이전트 + 실행권 = 치명적 공격면.

## 위임 계약 요약
- **🟢 자동 ≈ 18종** — 거의 다 *모니터링·분석·초안*. 에이전트가 매일 굴림.
- **🟡 승인 ≈ 10종** — 외부로 나가거나 제품을 바꾸는 것. 에이전트가 차려놓고 대표가 1클릭.
- **🔴 사람 ≈ 8종** — 돈·시크릿·권한·삭제·공개발행·배포. 영원히 사람.

## 가드레일 (전 등급 공통)
- 모든 액션 **감사 로그** + **킬 스위치**.
- 자격증명은 **읽기 전용 위주**, 🔴 리소스(결제·시크릿 키) 접근 자체를 차단.
- 🟡 승인은 **만료**(24h 미승인 시 폐기) + 묶음 일일 다이제스트.
- 에이전트 자기수정·자기권한상승 금지.

## 시작점 — Phase 1
**"매일 운영 브리핑" 에이전트**(🟢 읽기 전용만): A·B·C·E의 모니터링 항목을 모아 *오늘의 상태 + 유입/전환 + 할 일 Top3* 를 매일 아침 생성. 위험 0, 신뢰 적립 → Phase 2(🟡 콘텐츠 초안)로 확장.
