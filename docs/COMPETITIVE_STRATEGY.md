## 0. 결론 먼저

- **핵심 결정**: 가격을 내리지 않는다(Pro 19,900 유지). 대신 **기존 기능을 "입찰 안전망"이라는 하나의 약속 아래로 깊게** 모아 value를 올려 가격을 정당화한다. 가격 인하는 지투비플러스(월 9,900원)의 링으로 끌려가는 함정 — 우리가 이길 수 없는 싸움이다.
- **승리 이론(한 줄)**: *"월 10건씩 투찰하는 소규모 전문건설업자는 '더 따게 해준다'는 검증 불가능한 예측 약속보다, 무효투찰·적자낙찰·자격탈락·헛수고라는 확정 손실을 없애주는 검증 가능한 약속에 지갑을 연다."*
- **조직 원리**: 우리 브랜드("잃지 않게 지켜준다")를 슬로건에서 **측정 가능한 제품(입찰 안전망)**으로 전환. 경쟁사는 전부 "이기는 게임"(더 따게)을 팔고, 우리만 "지키는 게임"(한 번도 잃지 않게)을 판다 → 포지션이 아니라 **제품 카테고리** 차별화.
- **언더컷 기회(딥리서치 발견)**: 디마툴즈는 자격 사전진단을 **"기업 전용 AI" 연 990,000원**에 판다(전용모델 학습 5영업일). 우리는 이를 **기본 요금제에 즉시·무료로** 제공 → 가격이 아니라 실행력으로 이기는 언더컷.
- **미검증 전제(정직)**: 이 승리 이론의 지불의사는 딥리서치도 웹에서 증명하지 못했다. **오직 우리 자신의 데이터(진단 리드→가입→결제 전환)로만 증명 가능.** 그래서 안전망 구축과 별개로 전환 추적 훅이 모든 것의 전제다(§6).

## 1. 근거 — 딥리서치가 준 3개의 못

경쟁 3사는 서로 완전히 다른 게임을 한다. "경쟁사 공통 기본기" 같은 집합은 존재하지 않으므로, 3사 패리티는 세 개의 다른 전쟁을 동시에 치르는 함정이다.

| | 지투비플러스 | 디마툴즈 | 비드프로 |
|---|---|---|---|
| **게임** | 저가 정보제공 | AI 낙찰가 예측 | 컨설팅·대행(사람) |
| **가격** | 월 8,486~9,900원 | 월 33,000원 (+기업 전용 AI 연 99만) | 선납 66만~1,320만 + 성공보수 2.2% |
| **약속** | "다 보여줄게" | "찍어줄게" | "대신 해줄게, 못 따면 환불" |
| **채널** | 웹만 | 웹만(4개 조달 플랫폼 커버) | 웹만(레거시) |

딥리서치가 검증한 3개의 전략적 사실:

1. **예측은 신뢰할 수 없다 — 경쟁사 스스로 고백.** 디마툴즈는 자사 `/service-intro`에 "2025년 월평균 최고 예측율 약 65%"(실제 월별 54~64%)를 공시. 비드프로는 자사 낙찰확률 0.4~1.2%를 공개. → "우리는 예측 안 한다"는 정직 포지션의 최상급 근거.
2. **자격 사전진단에 지불의사가 있다 — 디마가 연 99만원에 판다.** 화이트스페이스는 '전무'가 아니라 **"기본 요금제 내 즉시 PASS/FAIL은 우리뿐"**으로 좁혀진다. 이건 축소가 아니라 언더컷 기회.
3. **무효투찰 고통은 실재한다 — 예측형 1위조차 인정.** 디마툴즈가 "마감·인증서·첨부 실수로 전체 무효"를 블로그 콘텐츠로 다룸. 단 **제품 기능이 아니라 콘텐츠**로만 → 우리가 제품으로 만들면 선점.

⚠️ **가격 앵커 리스크**: 디마툴즈 구조화 데이터에 "Lite 월 18,000원" 좀비 티어 잔존(현재 구매 불가). 부활 시 우리 Pro 19,900의 "40% 저렴" 앵커가 역전됨 → **차별화를 가격이 아닌 value(안전망+익스텐션)에 걸어야 하는 추가 근거.** (모니터링 대상)

## 2. 아키텍처 — 안전망 = 4대 확정 손실별 레이어

딥리서치에서 확정된 고객의 4대 확정 손실이 곧 제품 아키텍처다. 각 레이어는 "특정 손실을 잡는 그물"이며, 모든 기능은 어느 레이어에 속하는지로 정당화된다.

```
[손실1 무효투찰]  → ① 투찰 안전 게이트   (신규 · 안전망의 얼굴)
[손실2 적자낙찰]  → ② 안전 투찰가 밴드   (calculator 리포지션 · 예측 아님)
[손실3 자격탈락]  → ③ 자격 진단→처방     (QualificationChecker 확장 · 디마 99만원 언더컷)
[손실4 헛수고]    → ④ 반복낙찰 경보      (OpeningResult 재활용 · 들러리 식별)
                         │
                  [전달 채널] 크롬 익스텐션 오버레이(SAFE/CAUTION/DANGER 실시간)
                         └ 3사 전부 웹 단일 → 우리만 고객이 일하는 화면 위에 있음
```

### ① 투찰 안전 게이트 (신규)
디마가 *블로그로만* 다루는 무효투찰 예방을 *제품 기능*으로. 투찰 직전 5대 무효 사유(마감시각·인증서·필수첨부·금액 1원단위 절사·자격)를 체크리스트로 잠금. → **"Pro가 값하는 순간"**: 무효투찰 1회 회피 = 연 구독료 회수.

### ② 안전 투찰가 밴드 (기존 `calculator` 리포지션)
낙찰가 예측은 절대 안 함(§13 AI 원칙, 디마 65% 고백). 대신 **"이 밑으로 쓰면 적자/무효"인 안전선**을 시각적 밴드로. 예측 대신 "지켜야 할 선"을 판다 → 정직 포지션과 정합. 낙찰하한율 미만 = 무조건 DANGER(기존 도메인 로직 §10 그대로).

### ③ 자격 진단 → 처방 (기존 `QualificationChecker` 확장)
현재 PASS/FAIL 뱃지 → FAIL일 때 **"무슨 요건이 얼마 모자라서, 어떻게 채우면 통과하는지"** 처방까지. 디마 연 99만원 ML 진단(학습 5영업일)보다 **즉시·무료·실행가능**. 랜딩 비교표 한 줄: *"적격 진단 — BidEasy 기본 제공 vs 디마툴즈 연 99만원 별매."*

### ④ 반복낙찰 경보 — ❌ 폐기 (2026-07-19 게이트 판정)
~~"이 공고, 최근 N년 같은 업체가 가져갔어요."~~ **사전 등록 킬 기준(§6)에 따라 폐기**: 실측 결과 경보 대상 공고 0.00%(킬 기준 5%). 사정률 추첨 기반 경쟁입찰에서는 낙찰자가 준랜덤이라 반복낙찰 자체가 구조적으로 희박 — 디마의 "연례공고 낙찰 분석"은 수의계약류에나 유효할 가능성. 게이트가 무가치한 기능 개발을 막았다. 안전망은 ①②③ 3레이어로 확정.

## 3. 해자 — 왜 경쟁사가 못 베끼나 (가격 유지의 진짜 근거)

가격 유지의 근거는 기능 수가 아니라 **베끼기 어려운 자산**이다. 딥리서치가 확인한 방어선 3개:

1. **크롬 익스텐션 = 유통 해자.** 3사 전부 웹 단일 채널. 우리만 나라장터 화면 위에 있다. 안전망 4레이어를 익스텐션 오버레이로 띄우면 경쟁사는 유통 채널부터 새로 지어야 함.
2. **비예측 브랜드 = 정직 해자.** 예측형(디마·비드프로)은 자기 상품을 부정해야 하므로 "우리는 예측 안 한다"로 신뢰 있게 피벗 불가. 우리만 진심으로 말할 수 있음.
3. **개찰 데이터 플라이휠 = 데이터 해자.** 자가보정(주간 Celery)이 도는 한 안전선은 시간이 갈수록 정교해짐. 신규 진입자가 못 따라잡는 시간 축적.

## 4. 가격 유지의 언어 — value metric 재정의

"19,900 vs 9,900" 비교를 고객이 하게 두면 진다. 프레임을 **손실 회피 = 구독료 회수**로 전환:

> "무효투찰 한 번, 적자낙찰 한 번이면 1년 구독료가 그 자리에서 회수됩니다."

- KPI 원칙 정합: **낙찰률 사용 금지**(§ 함정목록 2). 핵심 지표 = **유효율(무효·적자 0)**. 안전망은 유효율 95%를 **눈에 보이게** 만드는 제품이다.
- 랜딩 반영: 경쟁 비교표에 "적격 진단 기본 제공 vs 별매" 축 + "손실 회피 = 구독료 회수" 카피. 기존 비예측·정직 포지션(§1.1 랜딩 개편) 유지.

## 6.5 낙찰 도달 벤치마크 결과 반영 (2026-07-17 — `docs/BENCHMARK_WIN_REACH.md`)

창업자 문제 제기("낙찰은 최종 목표 — 타사 동등 이상 모델 가능한가")에 대한 실측 판정 완료:
- **공식 판정 G3(포지션 유지)** — 단, 결정적 발견: **적격심사제에서 현 자가보정 전략이 이미 이론 상한의 92% 달성**(2025 win 41.5% vs oracle 45.3%). "더 정확한 새 모델"의 여지 자체가 작음 — 격차는 모델이 아니라 **노출**(`recommend_bid_price` API 미노출)에 있다.
- 디마 반박 탄약 갱신: 우리 표본 4,848건 기준 단일 전략 사후 최적조차 최근 레짐 37%(적격 45%) 상한 — 63~65%는 전 공고 대상 지표일 수 없음. ⚠️ "25% 상한" 주장은 레짐 분해 결과 사용 금지.
- "전략 투찰(안전/균형/공격 프리셋)" Pro+ 제품화는 **2026 데이터 400건+ 누적 후 G2 재판정을 거쳐 착수**(사전 등록 게이트 준수).
- 부수 수습 완료: 합성데이터 공개 엔드포인트 제거(정직 해자 보수) + 공사 하한율 2026 개정 티어드 반영(안전망 레이어 ② 정확도 보수).

## 7. 미해결·후속 (딥리서치 커버리지 공백)
