# Flutter 설치 및 환경 설정 가이드

현재 발생하는 `'flutter'은(는) 내부 또는 외부 명령...` 오류는 컴퓨터에 **Flutter SDK**가 설치되어 있지 않거나, 환경 변수(PATH)에 등록되지 않아서 발생합니다.

이 앱을 실행하려면 다음 단계에 따라 Flutter를 설치해야 합니다.

## 1. Flutter SDK 다운로드
1. [Flutter 공식 홈페이지 (Windows Install)](https://docs.flutter.dev/get-started/install/windows)에 접속합니다.
2. **"Install the Flutter SDK"** 섹션에서 최신 `flutter_windows_vX.X.X-stable.zip` 파일을 다운로드합니다.
3. 압축을 풉니다. (예: `C:\src\flutter` 위치에 압축 해제 권장)
   * **주의:** `Program Files` 처럼 권한이 필요한 폴더에는 설치하지 마세요.

## 2. 환경 변수 (PATH) 설정
Flutter 명령어를 어디서든 실행할 수 있도록 설정합니다.

1. 윈도우 검색창에 **"시스템 환경 변수 편집"**을 검색하고 실행합니다.
2. **[환경 변수]** 버튼을 클릭합니다.
3. **'사용자 변수'** 목록에서 `Path`를 찾아 더블 클릭합니다.
4. **[새로 만들기]**를 누르고, 아까 압축 푼 `bin` 폴더 경로를 입력합니다.
   * 예: `C:\src\flutter\bin`
5. 모든 창에서 [확인]을 눌러 저장합니다.

## 3. 설치 확인
1. 실행 중인 모든 명령 프롬프트(CMD)나 터미널을 껐다가 다시 켭니다.
2. 다음 명령어를 입력합니다:
   ```cmd
   flutter --version
   ```
3. 버전 정보가 잘 뜨면 설치 성공입니다!
4. 이제 `run_frontend.bat`를 다시 실행해보세요.

---

## 4. 백엔드 실행을 위한 Python 설치 (현재 오류 원인)
방금 보내주신 오류(`Python was not found`)는 **백엔드 서버**를 실행할 도구가 없어서 발생했습니다.

1. [Python 공식 다운로드 페이지](https://www.python.org/downloads/)로 이동합니다.
2. **"Download Python 3.x.x"** 노란색 버튼을 클릭합니다.
3. 설치 파일을 실행하고, **가장 중요한 단계!**
   * 설치 창 하단에 있는 **✅ Add Python to PATH** 체크박스를 **반드시 체크**해야 합니다.
   * (이걸 안 하면 명령어를 못 찾습니다)
4. [Install Now]를 눌러 설치를 완료합니다.

## 5. "Access Denied" 또는 "Failed to delete" 오류 해결법
`run_frontend.bat` 실행 시 **"파일을 삭제할 수 없다"**거나 **"Symlink"** 관련 오류가 뜬다면 다음 3가지를 순서대로 시도해보세요.

### 1단계: 윈도우 개발자 모드 켜기 (강력 추천) ⭐
Flutter가 윈도우 앱을 만들 때 '심볼릭 링크'라는 특수 기능을 사용하는데, 보안 때문에 막혀있는 경우가 많습니다.
1. 윈도우 설정 > **시스템** > **개발자용** (또는 검색창에 "개발자 설정" 검색)
2. **개발자 모드**를 **[켬]**으로 변경합니다.
3. (중요) 컴퓨터를 **재부팅**합니다.

### 2단계: 프로젝트 폴더 이동 (OneDrive 탈출)
**OneDrive** 폴더 안에 프로젝트가 있으면, OneDrive가 파일을 계속 동기화하느라 '잠금(Lock)'을 걸어서 오류가 발생합니다.
*   **해결:** `C:\Users\hosic\OneDrive\코딩\...` 에 있는 프로젝트 폴더 전체를
*   `C:\Projects\BidEasy` 처럼 **OneDrive 바깥으로 복사**해서 실행해보세요.

### 3단계: 관리자 권한으로 실행
1. `run_frontend.bat` 파일을 우클릭합니다.
2. **"관리자 권한으로 실행"**을 클릭합니다.
