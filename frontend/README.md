# BidEasy Frontend

## Setup

Since the `flutter` command was not detected in the environment, the project structure could not be automatically generated.

Please run the following command in this directory to initialize the Flutter project:

```bash
flutter create . --org com.hosicompany --project-name bideasy_app
```

## Structure
- `lib/`: Main application code
- `assets/`: Images and fonts (Pretendard)
