Chrome 익스텐션 설치

BidEasy 는 Chrome 익스텐션으로 동작합니다. Chrome 웹 스토어에서 1초 만에 설치할 수 있어요.

  • 아래 "Chrome 에 설치" 버튼 클릭
  • "Chrome 에 추가" → "확장 프로그램 추가" 클릭
  • 브라우저 우상단에 BidEasy 아이콘 표시되면 완료
💡 팁: 아이콘이 안 보이면 우상단 퍼즐 모양(🧩) 클릭 → "BidEasy" 옆 압정 모양(📌) 클릭하여 고정.
2

회원가입 + Pro 14일 무료 체험

신용카드 등록 없이 가입만 하면 Pro 모든 기능을 14일간 무료로 사용할 수 있습니다.

  • 회원가입 페이지에서 이메일·비밀번호로 가입
  • 또는 카카오·네이버 OAuth 로 1초 가입 (준비 중)
  • 가입 즉시 Pro 14일 자동 활성화 (AI 분석 일 50회, Deep Analysis 등)
  • 만료 후 자동으로 Free 요금제 전환 — 자동 청구 X
🎁 첫 달 50% 할인: 체험 만료 7일 이내 결제 시 첫 달 7,450원 자동 적용 (Pro 기준).
3

나라장터 공고 페이지 진입

BidEasy 는 나라장터(g2b.go.kr)의 공고 상세 페이지에서 자동으로 정보를 추출합니다.

  • 나라장터(www.g2b.go.kr) 접속
  • "입찰공고" 메뉴에서 관심 공고 클릭
  • BidEasy 가 공고번호·기초금액·마감일 자동 인식
⚠️ 주의: 나라장터 외 다른 사이트(예: 전자조달시스템 단일창구)에서는 자동 인식이 안 됩니다. 향후 확장 예정.
4

사이드패널에서 사정률·투찰가 계산

브라우저 우상단 BidEasy 아이콘 클릭 → 사이드패널이 열리고 즉시 투찰가 계산기가 표시됩니다.

  • 사정률 슬라이더: -2% ~ +2% 조정 → 투찰가 실시간 변경
  • 1원 단위 정확: A값(국민연금·건강보험 등) 자동 반영
  • 안전도 표시: 하한선 근접 시 ⚠️ 경고, 하한 미달 시 🔴 위험
  • 예정가격 범위: 기초금액 ±3% 자동 표시
💡 자가보정: BidEasy 는 매주 4,800+ 실 개찰 결과를 학습해 사정률 추천값을 자동 최적화합니다. 사용자는 슬라이더 손댈 필요 없음.
5

AI 공고 분석 (독소조항·위험요소)

"AI 분석" 탭을 누르면 GPT-4o-mini 가 공고문 전체를 읽고 핵심 정보를 자동 추출합니다.

  • 3줄 요약: 공고 핵심을 30초에 파악
  • 독소조항 탐지: 이행보증·하자보수·지체상금 등 위험 조건 자동 표시
  • 자격 요건 추출: 면허·지역·시공능력 한눈에
  • 마감·개찰 시각: 자동 정리
💡 비용: Free 일 1회 / Pro 일 50회 / Pro+ 무제한. Deep Analysis(첨부 HWP·PDF)는 Pro+ 전용.
6

즐겨찾기·마감 알림 설정

관심 공고를 즐겨찾기에 담아두면 마감 1시간 / 15분 전 Chrome 푸시 알림이 자동 발송됩니다.

  • "추적" 탭에서 별표(⭐) 아이콘 클릭 → 즐겨찾기 등록
  • 마감 임박 시 자동 Chrome 알림
  • 알림 클릭 → 해당 공고 페이지로 즉시 이동
⚠️ 권한: 처음 알림 발송 시 Chrome 이 권한 요청. "허용" 클릭 필요.
7

프로필 등록 (자격 자동 매칭)

회사 정보(면허·지역·시공능력 등)를 한 번 등록하면, 모든 공고 페이지에서 참가 가능 여부가 자동 표시됩니다.

  • "프로필" 탭에서 회사 정보 입력
  • 면허 (예: 전기공사업, 토목건축공사업)
  • 지역 (영업 지역 — 전국 또는 특정 시·도)
  • 시공능력평가액 (참여 가능 공사 규모 자동 필터)
💡 자동 매칭: 입력 후 공고 페이지 진입 시 사이드패널 상단에 "✓ 참여 가능" 또는 "✗ 자격 부족" 표시.

자주 묻는 질문

무료로도 충분히 쓸 수 있나요?

네. Free 요금제에서도 투찰가 계산기·공고 피드·즐겨찾기는 무제한 사용 가능합니다. AI 분석만 일 1회로 제한되며, 그 외 Deep Analysis·경쟁 강도 예측·기관 프로파일링 등은 Pro/Pro+ 전용입니다.

BidEasy 가 내 결제 정보를 저장하나요?

저장하지 않습니다. 모든 결제는 토스페이먼츠가 처리하며, BidEasy 는 토스로부터 결제 식별자(orderId, paymentKey)만 받습니다. 카드번호·CVC 는 일절 보관하지 않습니다.

AI 분석 결과는 얼마나 정확한가요?

GPT-4o-mini 기반 분석으로 공고문 핵심·독소조항·자격 요건 추출에 90% 이상 정확도를 보입니다. 다만 참고용으로 활용해주세요 — 최종 입찰 의사결정은 본인 확인 후 진행 권장.

나라장터 외 다른 사이트는 지원하나요?

현재는 나라장터(g2b.go.kr)만 지원합니다. 전자조달시스템 단일창구·국방전자조달 등 추가 사이트는 출시 후 점진 확장 예정입니다.

사정률 추천값은 어떻게 산출되나요?

BidEasy 의 자가보정 알고리즘이 매주 누적된 4,800+ 실 개찰 결과를 학습해 입찰방법·금액대별 최적 사정률을 자동 산출합니다. 사용자가 손대지 않아도 시간이 갈수록 정확도가 향상됩니다. (현재 운영 성과: 낙찰률 12.23%, 하한선 통과율 94.88%)

회사 정보를 입력하지 않아도 사용 가능한가요?

네. 모든 기능은 회사 정보 없이도 사용 가능합니다. 다만 "자격 자동 매칭"은 면허·지역·시공능력 정보가 있어야 정확합니다.

모바일에서도 쓸 수 있나요?

현재는 Chrome 브라우저(데스크톱) 전용입니다. 나라장터 입찰 자체가 모바일 미지원이라 데스크톱 우선 개발. 모바일 앱은 향후 검토 중.

문제 해결

사이드패널이 안 열려요
Chrome 우상단 BidEasy 아이콘 클릭. 안 보이면 퍼즐 아이콘(🧩) 클릭 → BidEasy 옆 압정(📌) 모양 클릭하여 고정.
공고 정보가 자동 인식 안 돼요
나라장터 공고 상세 페이지에서만 동작합니다. 목록 페이지에서는 인식 안 됨. 또한 페이지 로드 완료 후 1~2초 기다려주세요.
AI 분석이 "오류" 라고 나와요
일일 한도 초과(Free 1회/Pro 50회) 일 수 있습니다. 마이페이지에서 잔여 횟수 확인. 또는 OpenAI API 일시 장애 — 잠시 후 재시도.
로그인이 자꾸 풀려요
JWT 토큰 만료(60일) 일 수 있습니다. 다시 로그인해주세요. 만약 5분 만에 풀리면 브라우저 시크릿 모드 사용 중일 가능성 — 정상 모드 권장.
"권한이 필요해요" 라는 메시지가 떠요
Free 한도(AI 분석 일 1회) 초과이거나 Pro 전용 기능(Deep Analysis 등) 호출 중. 요금제에서 Pro 14일 체험 시작 가능.
익스텐션이 사라졌어요
Chrome 설정(chrome://extensions) 에서 BidEasy 확인. 비활성화 상태면 토글 ON. 삭제했으면 Chrome 웹 스토어에서 재설치.
